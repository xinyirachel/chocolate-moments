# **Todo App**
A chocolate theme CRUD app Todo App with database!

### **Git**
- [Repository for deployment](https://github.com/xinyirachel/chocolate-moments)

## **Functionality / features**

- **Create a post**

- **Read all posts**

- **Update posts**

- **Delete posts**

- **Sending information to developer**


## **Tech stack**

- JavaScript
- HTML/CSS
- ReactJS
- React Bootstrap
- Axios
- ExpressJS
- MongoDB
- Mongoose
- NodeJS
- Heroku
- Jest
